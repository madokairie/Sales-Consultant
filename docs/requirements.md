# Sales Consultant — 要件定義書

## 1. プロダクト概要

| 項目 | 内容 |
|------|------|
| アプリ名 | Sales Consultant |
| 目的 | 個別相談（1on1セールス）の成約率を最大化する |
| 対象ユーザー | プロモーター / セールスコンサルタント / コーチ / 講座運営者 |
| 利用形態 | ブラウザアプリ（PC・タブレット）。ローカルまたはサーバーで動作 |

## 2. 解決する課題

| 課題 | 本アプリでの解決策 |
|------|-------------------|
| セールスが苦手で何を話せばいいかわからない | AIが商品情報から9フェーズの相談スクリプトを自動生成 |
| 断り文句が来ると頭が真っ白になる | 反論切り返し辞典で「そのまま使えるセリフ」を事前に準備 |
| 面談中にスクリプトを見失う | 面談中カンペ機能で1ステップずつガイド表示 |
| 商談の振り返りが感覚的になる | AI分析が16項目・100点満点で客観評価 + 改善セリフ提示 |
| クライアントへの報告資料が作れない | スクリプト / 分析 / 反論辞典すべてPDFダウンロード可能 |
| データが消えるのが怖い | バックアップ/復元機能でJSON書き出し・読み込み |

## 3. 機能一覧

### 3.1 プロジェクト管理
- プロジェクト作成（スクリプト設計 or 商談分析から開始可能）
- プロジェクト一覧表示（スクリプト済み / 分析件数・平均スコア / 成約率）
- プロジェクト削除
- データバックアップ（全プロジェクトをJSONエクスポート）
- データ復元（JSONインポート。既存は上書き、新規は追加）
- アプリ内使い方マニュアル（ヘッダーの「📖 使い方」からアクセス）

### 3.2 商品設定（Config）
- 項目ごとの個別入力（15項目）
- テキスト一括入力（セールスページのテキストをAIが解析して自動抽出）
- 事前教育レベル設定（教育済み / 説明済み / 事前説明なし）
  - スクリプト構成・分析基準・反論傾向がこの設定で大きく変わる

### 3.3 スクリプト生成
- 商品情報 + 事前教育レベルからAIが9フェーズのスクリプトを自動生成
- フェーズ: オープニング → ヒアリング → 深掘り → 理想の未来 → 解決策提示 → 実績・証明 → オファー → 反論処理 → クロージング
- 各ステップにセリフ（talkScript）、キー質問、次へ進むサイン、コツ、繋ぎフレーズを含む
- ワンタップコピー対応
- PDF出力対応（A4・表紙付き・プロ品質デザイン）
- 再生成可能

### 3.4 面談中カンペ（リアルタイムガイド）
- ダークテーマの全画面ガイド（Zoom横で見やすい）
- 1ステップずつ表示。←→キー or ボタンで移動
- タイマー機能（Tキーでトグル。経過時間 + 目安時間を表示）
- フェーズ進捗バー + ステップ進捗バー
- セリフを大きめ文字で表示 + キー質問 / 次へ進むサイン / コツをコンパクト表示

### 3.5 反論切り返し辞典
- 商品情報 + 事前教育レベルからAIが7カテゴリの反論パターンを生成
- カテゴリ: 価格 / 時間 / 自信 / 家族 / 検討 / 競合 / タイミング
- 各反論に: 本当の心理 / 具体的な切り返しセリフ（間の指示付き）/ NGフレーズ（理由付き）/ やってはいけない行動 / フォローアップ質問
- カテゴリフィルター
- ワンタップコピー対応
- PDF出力対応
- 再生成可能

### 3.6 商談分析
- 文字起こしテキストの入力 or YouTube動画URLから字幕自動取得
- 商談結果（成約 / 失注 / 保留）の記録
- AI分析（16項目評価、100点満点）
  - 「導く面談」vs「売り込む面談」の判定
  - 4つの失敗パターン検出（初めて説明 / 雑談で終了 / 売ろうとしている / 決断を委ねている）
  - 決断確認ポイントの有無と対応評価
  - フェーズ別スコア
  - 改善点に「次回使える具体的なセリフ例」を付与
  - 見逃した機会に「こう言えばよかった」セリフを付与
- ワンタップコピー対応
- PDF出力対応
- 再分析可能

## 4. 技術要件

### 4.1 技術スタック
| 項目 | 技術 |
|------|------|
| フレームワーク | Next.js 16 (App Router) |
| 言語 | TypeScript |
| スタイリング | Tailwind CSS v4 |
| AI | Anthropic Claude API (claude-sonnet-4-20250514) |
| YouTube字幕 | youtube-transcript パッケージ |
| データ保存 | localStorage（ブラウザ内） |

### 4.2 外部依存
| サービス | 用途 | 必須 |
|---------|------|------|
| Anthropic API | スクリプト生成、商談分析、反論辞典生成、テキスト解析 | 必須 |
| YouTube | 字幕取得（YouTube動画URLからの文字起こし） | 任意 |
| Google Fonts | Noto Sans JP フォント | 推奨 |

### 4.3 環境要件
| 項目 | 要件 |
|------|------|
| Node.js | v18 以上 |
| ブラウザ | Chrome / Safari / Firefox / Edge（最新版） |
| API キー | Anthropic API キー（`ANTHROPIC_API_KEY`） |

## 5. セキュリティ

| 項目 | 対応 |
|------|------|
| データ保存場所 | ブラウザのlocalStorage（サーバーには送信しない） |
| API通信 | Anthropic APIへのサーバーサイド通信のみ（APIキーはサーバー側で保持） |
| 個人情報 | 商談の文字起こしテキストはAnthropic APIに送信される。個人名の取り扱い注意 |
| バックアップ | JSONファイルとしてローカルに保存。暗号化なし |

## 6. 制約事項

- データはブラウザのlocalStorageに保存されるため、ブラウザのデータ消去で消失する → バックアップ機能を推奨
- 1回のAPI呼び出しで処理するため、非常に長い文字起こし（12,000文字超）は切り詰められる
- YouTube字幕取得は字幕が存在する動画のみ対応
- PDF出力はブラウザの印刷機能（Ctrl+P / Cmd+P）を利用

```
    ├── AI分析
    └── PDF保存
```
