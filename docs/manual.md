## 1. 導入手順（セットアップ）

### 必要なもの

- パソコン（Mac / Windows）
- Node.js v18以上（[nodejs.org](https://nodejs.org/) からダウンロード）
- Anthropic APIキー（[console.anthropic.com](https://console.anthropic.com/) で取得）

### インストール手順

**ステップ1: ZIPを解凍**

配布された `salesconsultant.zip` を任意の場所に解凍してください。

**ステップ2: APIキーを設定**

解凍したフォルダの中に `.env.local` ファイルを作成し、以下の内容を書きます：

```
ANTHROPIC_API_KEY=sk-ant-あなたのAPIキーをここに貼り付け
```

> APIキーは [console.anthropic.com](https://console.anthropic.com/) の Settings → API Keys から取得できます。

**ステップ3: 起動**

ターミナル（Mac）またはコマンドプロンプト（Windows）を開き、以下を実行：

```bash
cd 解凍したフォルダのパス
npm install
npm run dev
```

**ステップ4: ブラウザで開く**

ターミナルに表示されたURL（通常 `http://localhost:3000`）をブラウザで開きます。

> ポートを変更したい場合: `npm run dev -- -p 3905` のように指定

## 2. 初めて使うとき

### トップページ

起動すると、2つの入口が表示されます：

- **📝 スクリプト設計** — 商品情報を入力してスクリプトを作る
- **📊 商談分析** — 商談の振り返りをAIで分析する

どちらからでも始められます。プロジェクト名を入力して「作成して始める」を押してください。

### アプリ内マニュアル

ヘッダー右上の「📖 使い方」ボタンをクリックすると、アプリ内に操作マニュアルが表示されます。初めて使う方はここから各機能の使い方を確認できます。

### プロジェクトとは？

1つの商品・サービスに対して1つのプロジェクトを作ります。

例：
- 「コーチングプログラム個別相談」
- 「オンライン講座セールス」
- 「Aさん案件 - ダイエット講座」

## 3. スクリプトを作る

### 3-1. 商品情報を入力する

「⚙️ 設定」タブで商品情報を入力します。

**入力方法は2つ：**

| 方法 | 使いどころ |
|------|-----------|
| 項目ごとに入力 | 正確に入力したいとき |
| テキストから一括入力 | セールスページやLPのテキストを貼り付けてAIに抽出させる |

**重要な設定項目：**

| 項目 | 説明 |
|------|------|
| 事前教育レベル | スクリプトの構成が大きく変わる最重要設定 |
| よくある断り文句 | 入力すると反論処理フェーズに反映される |
| ゴール | 「その場で成約」「次のステップ」等で構成が変わる |

**事前教育レベルの違い：**

| レベル | 面談での説明量 | 成約率目安 |
|--------|-------------|----------|
| 教育済み（価格まで公開） | 商品説明不要。確認メイン | 80-90% |
| 説明済み（価格未公開） | 軽く確認 + 価格提示が重要 | 40-60% |
| 事前説明なし | フル説明が必要 | 10-30% |

### 3-2. スクリプトを生成する

設定画面の右上「📝 スクリプト生成」ボタンを押すと、AIが9フェーズのスクリプトを生成します（30秒〜1分）。

**9フェーズ構成：**

1. 👋 オープニング（場を整える、ゴール共有）
2. 👂 ヒアリング（15分以内に課題を引き出す）
3. 🔍 課題の深掘り（痛みの明確化）
4. ✨ 理想の未来（解決後のビジョン）
5. 💡 解決策の提示（商品紹介）
6. 📊 実績・証明（Before/After）
7. 🎯 オファー提示（価格・条件）
8. 🛡️ 反論処理（断り文句への対応）
9. 🤝 クロージング（決断確認 → 申込 or お見送り）

### 3-3. セリフをコピーする

各ステップのセリフ右上に「コピー」ボタンがあります。タップするとクリップボードにコピーされます。

## 4. 面談中カンペを使う

スクリプトタブの「🎯 面談カンペ」ボタンを押すと、面談中に使えるガイド画面が開きます。

### 使い方

1. ZoomやGoogleMeetの横にブラウザウィンドウを配置
2. 面談開始時に「▶」ボタンまたは「T」キーでタイマー開始
3. セリフを読みながら、「→」キーまたは「次へ →」ボタンで進む
4. キー質問や「次へ進むサイン」を確認しながら進行

### ショートカット

| キー | 動作 |
|------|------|
| → or スペース | 次のステップ |
| ← | 前のステップ |
| T | タイマー開始/停止 |

### ポイント

- 暗い背景なのでZoom映像と区別しやすい
- セリフは大きめ文字で表示
- 上部のフェーズバーをクリックすると任意のフェーズにジャンプ

## 5. 反論切り返し辞典を作る

「🛡️ 反論辞典」タブで「反論辞典を生成」ボタンを押します。

### 7カテゴリの反論パターン

| カテゴリ | 代表的な断り文句 |
|---------|----------------|
| 💰 価格 | 「ちょっと高いですね」「予算が...」 |
| ⏰ 時間 | 「今忙しくて」「時間がない」 |
| 💪 自信 | 「自分にできるか不安」 |
| 👨‍👩‍👧 家族 | 「夫/妻に相談したい」 |
| 🤔 検討 | 「考えさせてください」 |
| 🔄 競合 | 「他のも見てみたい」 |
| 📅 タイミング | 「もう少し後で」 |

### 各反論カードの見方

| セクション | 内容 |
|-----------|------|
| 本当の心理 | この断り文句の裏にある本音 |
| 切り返しフレーズ | そのまま使える具体的なセリフ（間の指示付き） |
| NGフレーズ | 絶対に言ってはいけない言葉（理由付き） |
| やってはいけない行動 | その場でやりがちなNG行動 |
| フォローアップ質問 | 反論処理後に投げかける質問 |

### 大原則

> **押さない・追わない・丁寧にお見送り**
>
> NOと言われたら追わない。押すとキャンセル・クレーム・返金になる。
> 押さないと、相手は安心して本音を話し、自分から「やりたい」と言う。

## 6. 商談を分析する

### 6-1. 商談を記録する

「📊 商談記録」タブで「+ 商談を記録」を押し、以下を入力：

- お客様名
- 結果（成約 / 失注 / 保留）
- 文字起こしテキスト

**文字起こしの入手方法：**

| 方法 | やり方 |
|------|--------|
| YouTube動画URL | URLを貼って「字幕を取得」ボタン（字幕がある動画のみ） |
| Zoom文字起こし | Zoomの文字起こし機能で出力したテキストを貼り付け |
| 手動入力 | 商談メモや議事録を貼り付け |

### 6-2. AIで分析する

保存した商談の「🔍 AI分析」ボタンを押すと、AIが以下を分析します：

- **総合スコア**（100点満点）
- **「導く面談」判定** — 売り込みになっていないかチェック
- **失敗パターン検出** — 4つの致命的パターン（初回説明 / 雑談で終了 / セールス / 決断委ね）
- **フェーズ別評価** — 各フェーズごとのスコアとコメント
- **改善点** — 具体的な改善セリフ例付き
- **見逃した機会** — 「こう言えばよかった」セリフ付き
- **次回のアクション** — やるべきことリスト

### 分析のポイント

- 商品情報を設定しておくと分析精度が上がる（未設定でも分析可能）
- 「失注」の場合、失注原因の詳細分析が追加される
- 「保留」の場合、「後で」を防ぐ設計の診断が追加される

## 7. PDFをダウンロードする

各タブの「PDF保存」ボタンを押すと、プロ品質のPDFが生成されます。

| 種類 | 内容 |
|------|------|
| スクリプトPDF | 表紙 → タイムライン → 各フェーズ詳細 |
| 反論辞典PDF | 表紙 → 基本原則 → 各反論詳細（NGフレーズ付き） |
| 分析レポートPDF | 表紙 → スコア・総評 → フェーズ評価 → 改善点 |

**PDF保存の手順：**

1. 「PDF保存」ボタンをクリック
2. 印刷プレビューが開く
3. 「送信先」を「PDFとして保存」に変更
4. 「保存」をクリック

> Chromeの場合、「詳細設定」→「背景のグラフィック」にチェックを入れると色が反映されます。

## 8. データのバックアップと復元

### バックアップ

トップページ右上の「バックアップ」ボタンをクリックすると、全プロジェクトのデータがJSONファイルとしてダウンロードされます。

ファイル名例: `sales-consultant-backup-2026-03-13.json`

**含まれるデータ：**
- 全プロジェクトの設定情報
- 生成したスクリプト
- 反論切り返し辞典
- 商談記録と分析結果

### 復元

1. トップページ右上の「復元」ボタンをクリック
2. バックアップしたJSONファイルを選択
3. 「インポート完了: X件追加、Y件更新」と表示されれば成功

> 同じIDのプロジェクトは上書き、新しいプロジェクトは追加されます。

### バックアップの推奨タイミング

- 週に1回
- 大量の商談分析を行った後
- ブラウザのキャッシュをクリアする前
- 別のPCで使い始めるとき

## 9. クライアント案件での使い方

### 案件ごとのプロジェクト運用

```
プロジェクト例：
├── Aさん案件 - コーチングプログラム
│   ├── 設定: 商品情報入力済み
│   ├── スクリプト: 生成済み → Aさんに共有
│   ├── 反論辞典: 生成済み → Aさんに共有
│   └── 商談分析: 5件（平均72点）
├── Bさん案件 - オンライン講座
│   └── ...
└── Cさん案件 - コンサル契約
    └── ...
```

### プロモーターとしての活用フロー

**STEP 1: 準備**
1. クライアントの商品情報をヒアリングして設定に入力
2. 事前教育レベルを正確に設定（最重要）
3. よくある断り文句があれば入力

**STEP 2: 納品物作成**
1. スクリプト生成 → PDF保存 → クライアントに共有
2. 反論辞典生成 → PDF保存 → クライアントに共有
3. 必要に応じて面談カンペの使い方を説明

**STEP 3: フォローアップ**
1. クライアントの商談録音/文字起こしを受領
2. 商談分析を実行
3. 分析レポートPDFをクライアントに共有
4. 改善点を一緒にレビュー
5. 必要に応じてスクリプトを再生成

### クライアントへの説明テンプレート

> このツールで、〇〇さんの商品に最適化された個別相談のスクリプトを作成しました。
> 9つのフェーズに分かれていて、各フェーズでどんなセリフを使うか、具体的なキー質問、次に進むタイミングまで入っています。
> また、よくある断り文句への切り返しフレーズ集もお渡しします。「押さない・追わない」原則で、お客様が安心して決断できる対応方法です。
> 面談後は録音の文字起こしをいただければ、AIが客観的に分析して改善点をお伝えします。

## 10. よくある質問

### Q: APIキーの料金はどれくらいかかる？

A: Anthropic APIは従量課金です。目安：
- スクリプト生成 1回: 約 $0.05〜0.10
- 商談分析 1回: 約 $0.05〜0.10
- 反論辞典生成 1回: 約 $0.10〜0.15

月に数十回使っても数ドル程度です。

### Q: データはどこに保存される？

A: ブラウザのlocalStorageに保存されます。サーバーには保存されません。ブラウザのデータを消去すると消えるので、定期的にバックアップしてください。

### Q: 文字起こしテキストはAIに送信される？

A: はい。商談分析時にAnthropic APIに送信されます。個人名や機密情報が含まれる場合はご注意ください。Anthropicはプロンプトをモデル訓練に使用しないポリシーです。

### Q: 別のPCでも使える？

A: はい。バックアップ機能でJSONファイルを書き出し、別のPCで復元すればデータを移行できます。

### Q: スクリプトが商品に合わない場合は？

A: 設定情報を詳しく入力するほど精度が上がります。特に「商品概要」「ベネフィット」「事前教育レベル」を充実させてください。それでも合わない場合は「🔄 再生成」で別バージョンを生成できます。

### Q: 複数のブラウザタブで開いても大丈夫？

A: データはlocalStorageで管理されているため、同時編集は推奨しません。1つのタブで操作してください。

### Q: オフラインで使える？

A: AI生成・分析機能はインターネット接続が必要です。既に生成されたスクリプトや分析結果の閲覧、面談カンペはオフラインでも利用可能です。
